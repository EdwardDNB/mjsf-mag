# Лабораторна робота №3. Використання Slots та Provide/Inject у Vue 3

## Мета

1. Ознайомитися з механізмом **slots** у Vue 3.
2. Навчитися використовувати **provide/inject** для передачі даних між компонентами.
3. Реалізувати приклад компонента **Tabs** з можливістю кастомізації відображення вкладок.

---

## Засоби та інструменти

- **Vue 3**
- **Tailwind CSS**
- **Node.js** + npm/yarn
- (опціонально) [Flowbite](https://flowbite.com/docs/components/tabs/#default-tabs) як приклад UI

---

## Завдання

### 1. Підготовка середовища
1. Використати проект із попередньої лабораторної роботи.
2. Переконатися, що Tailwind CSS налаштований.
3. Ознайомитися з прикладом Tabs із Flowbite.

---

### 2. Створення компонентів Tabs

#### 2.1. Компонент `Tabs`
- Використовує **provide**, щоб передати інформацію про активну вкладку дочірнім компонентам.
- Пропси:
    - `defaultTab`: ідентифікатор вкладки, яка активна за замовчуванням.
    - `variant`: стиль відображення (`underline`, `pills`, `boxed`).
- Має **slot** для вкладок (`Tab`) та контенту (`TabPanel`).
- Відповідає за логіку перемикання вкладок.

#### 2.2. Компонент `Tab`
- Отримує дані через **inject** від `Tabs` (активна вкладка та метод зміни активної).
- Відображає кнопку вкладки зі стилізацією Tailwind.
- Використовує **slot** для назви або іконки вкладки.

#### 2.3. Компонент `TabPanel`
- Отримує дані через **inject** від `Tabs`.
- Відображає вміст тільки якщо вкладка активна.
- Використовує **slot** для вмісту вкладки.

---

### 3. Варіанти стилізації
Реалізувати кілька стилів вкладок за допомогою Tailwind CSS:

- **Underline** – активна вкладка підкреслюється.
- **Pills** – закруглені кнопки.
- **Boxed** – вкладки з обрамленням.

Пропс `variant` у `Tabs` змінює стилізацію.

---

### 4. Демо-сторінка
Створіть сторінку з такими прикладами:

1. Tabs із трьома вкладками у стилі **underline**.
2. Tabs з іконками у стилі **pills**.
3. Tabs з кастомним вмістом у вкладках (**slot** для кнопки вкладки).

## Приклад використання

```vue
<!-- TabsDemo.vue -->
<template>
  <Tabs defaultTab="tab1" variant="underline">
    <!-- Заголовки вкладок -->
    <template #tabs>
      <Tab id="tab1">Профіль</Tab>
      <Tab id="tab2">Налаштування</Tab>
      <Tab id="tab3">Контакти</Tab>
    </template>

    <!-- Контент вкладок -->
    <template #panels>
      <TabPanel id="tab1">Вміст профілю...</TabPanel>
      <TabPanel id="tab2">Форма налаштувань...</TabPanel>
      <TabPanel id="tab3">Контактна інформація...</TabPanel>
    </template>
  </Tabs>
</template>
